The Visual Challenge

Translating heritage into a scalable digital interface.

The goal of the Holland Walking Tour app was to modernize physical tourism maps into an intuitive, visually stunning mobile experience. As the UI Designer, my primary objective was to establish a design language that honored the city's historic Dutch roots while adhering strictly to modern iOS Human Interface Guidelines.

I focused on creating a frictionless visual experience for users walking outdoors: prioritizing high-contrast UI elements, legible typography systems, and recognizable bottom-sheet navigation paradigms.

Foundation

Color & Typography Strategy

Designing for an app primarily used outdoors in direct sunlight dictates strict visual rules. I established a high-contrast palette anchored by Heritage Teal, ensuring active states and primary buttons passed WCAG AA accessibility standards against our surface backgrounds.

UI Components

Systematic UI Design

To ensure consistency across the application, I built a component library utilizing an 8pt grid system. This atomic approach allowed for rapid prototyping and ensured all paddings, margins, and touch targets (minimum 44x44px) were mathematically sound.

Media Player UI

Designed a custom audio player interface for the "Listen to Audio" feature. Used clear visual hierarchy for progress bars and playback controls, ensuring the user can easily scrub through historical narrations without visual clutter.

Map & Bottom Sheet

Implemented a dynamic bottom-sheet overlay for the map view. This allows users to retain context of their geographic location while smoothly swiping up to browse location details, adhering to familiar mobile interaction patterns.

The Impact

Design that Drives Engagement

As a UI Designer, my role goes beyond making things "look pretty." For the Holland Walking Tour, I architected a visual system that actively reduces cognitive load. By standardizing the component library, optimizing touch targets, and curating an accessible color palette, the interface empowers users to look up at the city's architecture, rather than squinting down at their phones.
